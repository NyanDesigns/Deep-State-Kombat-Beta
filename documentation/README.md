# Deep State Kombat Documentation

This folder contains a comprehensive breakdown of the project's architecture, folder structure, character details, and current adjustments/problems.

## Files

- [Architecture](./architecture.md): High-level system architecture overview and design patterns.
- [Core Management](./Core_Management.md): Comprehensive documentation of all core management systems (AppStateManager, SceneManager, RenderSystem, etc.).
- [Game System](./Game_System.md): Detailed documentation of all game systems (Fighter, CombatSystem, CollisionSystem, GameState, AIController).
- [UI and Systems](./UI_and_Systems.md): Complete documentation of UI components, animation systems, character management, utilities, and shaders.
- [Folder Structure](./folder_structure.md): Breakdown of the project directory and file organization.
- [Characters](./characters.md): Detailed statistics, configuration, and technical breakdown of character system.
- [Adjustments & Problems](./adjustments_problems.md): Current issues, debug tools, solved problems, and planned improvements.

## Project Overview

**Deep State Kombat** is a parody-themed fighting game built with Three.js. It features a system-based architecture that manages everything from 3D rendering and combat logic to UI and persistent storage.
